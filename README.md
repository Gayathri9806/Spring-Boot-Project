# Employee Management System

## Project Overview

Employee Management System is a RESTful web application developed using Spring Boot. 
This application performs CRUD (Create, Read, Update and Delete) operations on employee records stored in MySQL database.

---

## Technologies Used

* Java
* Spring Boot
* Spring Data JPA
* MySQL
* Maven
* Postman
* Git
* GitHub

---

## Project Architecture

Controller Layer

↓

Service Layer

↓

Repository Layer

↓

MySQL Database

---

## Features

* Create Employee
* Get All Employees
* Get Employee By ID
* Update Employee Details
* Delete Employee
* Exception Handling
* Database Connectivity

---

## API Endpoints

### Create Employee

POST /employees

### Get All Employees

GET /employees

### Get Employee By ID

GET /employees/{id}

### Update Employee

PUT /employees/{id}

### Delete Employee

DELETE /employees/{id}

---

## Database

MySQL database is used to store employee information.
